# Laravel Keycloak API

This project is a backend authentication API built with **Laravel** and integrated with **Keycloak** using **OpenID Connect (OIDC)** and **JWT validation**.

It demonstrates how to securely protect API routes using Keycloak-issued access tokens and return authenticated user data.
